# Bus Management System

This is a bus management system that allows admins to manage drivers, buses, and routes. It also allows drivers to start and end journeys, and mark stoppages.

## Installation

1. Clone the repository.
2. Install the dependencies.
3. Start the server.

```
git clone https://github.com/bus-management-system/bus-management-system.git
cd bus-management-system
npm install
npm start
```

## Usage

### Admins

Admins can login to the system using the `/admin/login` endpoint. Once logged in, they can perform the following tasks:

* Create new drivers
* Create new buses
* Create new routes
* Assign drivers to buses
* Assign buses to routes
* View a list of all drivers
* View a list of all buses
* View a list of all routes

### Drivers

Drivers can login to the system using the `/driver/login` endpoint. Once logged in, they can perform the following tasks:

* Start a journey
* End a journey
* Mark a stoppage

## Code Structure

The codebase is divided into the following directories:

* `api`: Contains the API endpoints.
* `database`: Contains the database models.
* `entities`: Contains the business logic.
* `middleware`: Contains the middleware.
* `routes`: Contains the routes.
* `services`: Contains the services.
* `utils`: Contains the utility functions.
